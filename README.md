# USANDO-DA-IA-PARA-BUSTAR-PRODUTIVIDADE-SQL

Nesta atividade o objetivo foi gerar um banco de dados com o auxílio da IA.
O *objetivo* é justamente mostrar que a *IA* pode *turbinar* a produtividade.
Sempre bom frizar que todos os dados, foram gerados por IA e são dados baseados em um anime.
Logo, ao se *tratar de dados empresariais*, eles teriam que ser inseridos um a um.
